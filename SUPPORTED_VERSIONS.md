# Supported Versions

CertiVault is pre-1.0 and under active development. Security and critical
maintenance fixes are applied to the latest maintained branch only.

| Version or branch | Supported | Notes |
| --- | --- | --- |
| `main` | Yes | Stable default branch for production-ready code. |
| `ecsoc-2026` | Yes | Active contribution branch for ECSoC work. |
| Tagged releases | Best effort | Support policy starts with the first public release. |
| Older commits and forks | No | Please update to a maintained branch before reporting issues. |

When CertiVault publishes its first stable release, this file should be updated
with explicit semantic-version support windows.
